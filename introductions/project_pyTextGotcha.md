## 프로젝트 개요

### Python으로 이미지속에서 세부 글자 추출

* OpenCV 를 이용하여 여섯 단계의 이미지처리를 통해 글자로 추정되는 부분을 찾기
* 딥러닝 Image Classification 을 통해서 찾은 부분이 글자인지 아닌지 판단
* OCR 엔진을 통해 글자이미지에서 TEXT 추출
* [GitHub](https://github.com/nicewoong/pyTextGotcha)
